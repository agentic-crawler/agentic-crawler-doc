PayPal Presentment Crawler

An automated crawler operated by PayPal to review merchant checkout and PayPal product presentment experiences for quality, availability, and compliance validation.

What it does

The crawler navigates live product pages using a browser agent and checks for required PayPal integration signals — buttons, SDK calls, and checkout flows — across target merchant surfaces.

What it checks

Button presence

PayPal / Venmo / Pay Later buttons rendered correctly on checkout pages

SDK integration

PayPal JS SDK loaded with the correct client ID and intent

Checkout flow

End-to-end flow completes without broken redirects or missing steps

Visual adherence

Button styling, placement, and branding match PayPal merchant guidelines

Validation

PayPal follows Cloudflare's request-signature verification process using the keys and configuration provided by Cloudflare. The crawler includes the required signature headers on all requests.
